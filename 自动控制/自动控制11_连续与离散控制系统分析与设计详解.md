# Examen 11 详细讲解与解答（两时间常数+纯延时：P/ZN-PI/频域PI与数字分析）

- 题意概览：第一段对象含两时间常数与纯延时，先给出微分方程、近似响应与波德、延时相位；将延时近似为时间常数，求闭环临界增益 `k_max` 与极限振荡，并给出 Ziegler–Nichols PI；随后在采样周期 `Te=τ` 下给出可采频率、不可控常数时间判别、离散模型与数字比例闭环稳定性、极点与最终值；第二段为无延时的两时间常数对象，依次进行 P 闭环二阶化、5% 超调整定与静差；再做 ZN-PI与频域 PI 设计（选择 `Ti=τ2` 与相位裕度45°的增益），最后进行采样化与数字 PI的跟踪误差分析。

## 1. 含纯延时的两时间常数对象（连续域）

对象：

$G(p)=\dfrac{A e^{-\tau p}}{(1+\tau_1 p)(1+\tau_2 p)}$，数值：`A=1, τ=10, τ1=1, τ2=100`。

- 微分方程：将两一阶串联写成

$(1+\tau_1 p)(1+\tau_2 p) Y(p) = A e^{-\tau p} W(p)$，回到时域得

$\tau_1 \tau_2 \ddot y + (\tau_1+\tau_2) \dot y + y = A w(t - \tau)$。

- 阶跃近似响应：两时间常数串联产生 S 形；纯延时导致响应整体右移 `τ`，之后按二阶近似逐步上升。

- 波德（幅频）：低频增益约 `A`，两拐点在 `1/τ1=1 rad/s` 与 `1/τ2=0.01 rad/s`，高频斜率 −40 dB/dec。延时带来相位近似 `−ω τ` 线性下斜。

- 纯延时在 `ω0=1/τ` 处的相位：`φ_delay(ω0)=−ω0 τ=−1 rad≈−57.3°`。

将延时近似为一一阶：$e^{-\tau p} \approx \dfrac{1}{1+\tau p}$，得到

$G_1(p)=\dfrac{A}{(1+\tau_1 p)(1+\tau_2 p)(1+\tau p)}$。

闭环比例 `k`：$H(p)=\dfrac{k G_1}{1+k G_1}$。

- 临界增益与极限振荡：在 Nyquist 或幅相法中寻找使开环相位为 −180° 且增益为 0 dB 的 `k_max`。该三一阶系统在极限时闭环出现一对纯虚共轭极点（实部为 0），与题面结论相符。

- 极限振荡周期：$T_u=2\pi/\omega_u$（以图上读到的极限交叉频率 `ω_u`）。据此给 ZN-PI：`K=0.45 k_max, Ti=T_u/1.2`。

## 2. 采样与数字比例闭环分析（`Te=τ`）

- 采样可测最大频率：`f_max=1/(2 Te)=1/(2 τ)`（奈奎斯特）。
- `Te=τ=10 s`，则 `f_max=1/20 Hz=0.05 Hz`，远小于 `1/τ2=0.01 rad/s≈0.0016 Hz` 的低频特征，导致较慢模态在采样下几乎不可控/不可观（题面指出 `τ2` 在该设置下不可控）。
- 在忽略 `τ2` 的近似下：$G_2(p)\approx \dfrac{A e^{-\tau p}}{1+\tau_1 p}$。

ZOH 离散：

$G(z)=\dfrac{(1-a) z^{-1}}{1 - a z^{-1}}$，其中 $a=e^{-T_e/\tau_1}$；若含延时 `τ`，再乘以 `z^{-d}`（`d=τ/Te=1`）。

比例闭环稳定性：闭环分母 `1 + k_{ech} G(z)` 应满足 Jury 条件，得到区间 `0<k_{ech}<k_{ech,max}`。在该近似下可快速给出基于 `a` 的解析界：`k_{ech,max}≈(1+a)/(K'·(1-a))`（需根据实际反馈系数拼接）。

- 极点位置与最终值：在 `k_{ech}=k_{ech,max}` 处计算闭环极点（题面给出一般表达式），最终值 `y(∞)=BF(1)`；无积分时对阶跃参考存在非零静差。

- 比较 `k_{ech}` 与 `k_{max}`：在本题设置与近似下可能出现 `k_{ech} > k_{max}` 的意外结果（题面点评“反常但源于假设”）。

## 3. 两时间常数对象（无延时）之 P 与 PI（连续域）

$G(p)=\dfrac{A}{(1+\tau_1 p)(1+\tau_2 p)}$，数值：`A=1, τ1=1, τ2=2`。

- 微分方程：$\tau_1 \tau_2 \ddot y + (\tau_1+\tau_2) \dot y + y = A w$；代入数值得 `2 \ddot y + 3 \dot y + y = w`（与题面一致）。
- 开环极点：`p1=−1, p2=−0.5`，开环稳定。

比例闭环 `k`：

$BF(p)=\dfrac{k}{2 p^2 + 3 p + 1 + k}$。

二阶标准型匹配：

$\omega_0=\sqrt{\dfrac{k+1}{2}},\ \xi=\dfrac{3}{2\sqrt{2(k+1)}}$。

- 5% 超调整定：由 $M_p=5\%$ 求出 `ξ≈0.707` 的近似或直接按题面公式给出 `k0≈1.36`。
- 静差：$e_{ss}=1/(1+k0)≈0.46$。

ZN-PI（由阶跃响应图读数）：题面给 `τ≈0.38, T≈4.0`，得 `K≈9.35, Ti≈1.27`。

频域 PI：选择 $Ti=τ2$ 使开环仅一处折点；再调 `K` 使相位裕度 45°，题面给 `K≈2.8184`（9 dB）。闭环静差为 0（积分器）。

## 4. 采样化与数字 PI 跟踪误差

采样周期：`Te=τ1/10`。两一阶串联 ZOH 离散的标准二阶形式：

$G(z)=\dfrac{b_1 z + b_0}{z^2 + a_1 z + a_0}$，题面数值：

$G(z)=\dfrac{0.002379 z + 0.002263}{z^2 - 1.856 z + 0.8607}$。

数字比例闭环：

$BF(z)=\dfrac{k_{ech}(b_1 z + b_0)}{z^2 + (a_1 + k_{ech} b_1) z + (a_0 + k_{ech} b_0)}$。

Jury 稳定域（题面结论）：`−1.0124 < k_{ech} < 61.5555`（实际取 `k_{ech}>0`）。

最终值：$y(∞)=BF(1)≈0.983$。

数字 PI 控制器：

$C(z)=k_p + k_i T_e \dfrac{z}{z-1}$，差分式：

$u_k = u_{k-1} + (k_i T_e + k_p) \varepsilon_k - k_p \varepsilon_{k-1}$。

单位斜坡输入的滞后误差：

$\varepsilon_t(∞)=\dfrac{1 + a_1 + a_0}{k_i(b_1 + b_0)}$（题面推导）。

## 5. 要点总结

- 含延时对象可用一阶近似进行整定与稳定性分析：临界增益对应纯虚闭环极点，并据此给 ZN-PI 参数。
- 采样周期过大将使慢模态不可控/不可观，需适当减小 `Te` 或使用预测补偿；离散比例闭环稳定域可由 Jury 快速求得。
- 无延时两时间常数对象的 P 闭环可标准二阶化并据此匹配阻尼/超调；频域 PI 设计可选 `Ti=τ2` 使折点唯一并调 `K` 达到目标相位裕度。
- 数字 PI 的递推式与跟踪阶跃/斜坡的误差公式便于快速评估性能与参数选择。
