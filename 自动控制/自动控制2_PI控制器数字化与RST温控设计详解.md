## 1 Régulation analogue de phase（PI）

已知：对象相位-频率传递函数以 `H(p)` 表示；给出 Evans 图（C.12）与开环/闭环结构（C.13）。

1) 题意解析（Ziegler–Nichols 闭环极限法求 PI）
- 目标：基于闭环等幅自激点（极限增益 Ku 与极限振荡周期 Tu）整定一个模拟 PI：
  - PI（ZN 闭环法）标准参数：
    - Kp = 0.45 Ku
    - Ti = Tu / 1.2
- 获取 Ku, Tu：在 Evans 或实测中逐渐增益上调，至闭环开始持续振荡时的增益与周期即为 Ku, Tu。

解题步骤
- 从 C.12（或配套实验/图）测得 Ku 与 Tu。
- 代入 ZN 公式：Kp = 0.45 Ku，Ti = Tu/1.2。
- 模拟 PI 为：C_PI(p) = Kp(1 + 1/(Ti p))。

2) 题意解析（读 Bode，测相位裕度与增益裕度）
- 给出“PI + H(p)”的 Bode 图（C.14/C.15）。
- 操作：
  - 增益交越频率 ω_gc 处相位 φ(ω_gc) → 相位裕度 PM = 180° + φ。
  - 相位交越频率 ω_pc（相位 = −180°）处的幅值 → 增益裕度 GM（到 0 dB 的差）。

解题步骤
- 在图上读出 ω_gc 及相位；在 ω_pc 读出幅值。
- 给出 PM 与 GM 数值（保留至 10^{-3}）。

3) 题意解析（超过冲太大 → 增加相位裕度）
- 要求把 PI 的整体增益降低，使闭环相位裕度达到 90°（Ti 不变）。
- 方法：按 Bode 上移动幅线（整体下降）直至 PM ≈ 90°；读出对应的新增益缩放系数 k_new/k_old。

解题步骤
- 在 C.14/C.15 上：
  - 找到达到 PM = 90° 的 ω_gc_new；
  - 将幅线下移至该点 0 dB，得到新 Kp' = α Kp；
  - 给出 Kp'，Ti 不变。

4) 题意解析（经验判断阶跃响应形状）
- 相位裕度约 90° 的闭环通常：
  - 超调很小（近 0），响应较钝但稳定余度大；
  - 上升时间与带宽相权衡，适合“稳”优先。

解答要点
- 简述：PM≈90° → 小超调、较慢但鲁棒。

## 2 Transposition du régulateur analogique（数字化）

1) 题意解析（数字化优势）
- 目标：把模拟 PI 迁移为数字控制。
- 优点（两点）：
  - 灵活性与可重构：参数自适应、非线性/逻辑易实现、易于诊断与记录。
  - 抗漂移与复现性：模拟元件温漂/老化影响小；可加入防饱和、限幅与滤波。

2) 题意解析（方法选择 + 推导）
- 任选合适的离散化方法并简述理由：
  - 常用：Tustin（双线性，频域映射一致性好）、前向/后向差分（简单，但高频失真大）、零极点匹配（适于整形）。
- 以 Tustin 为例：
  - 双线性：p ≈ (2/T_e) (1 − z^{-1})/(1 + z^{-1})
  - 对模拟 PI：C_PI(p) = Kp(1 + 1/(Ti p)) ⇒ 写成同分母后代换 p，化简得 C_PI(z)。

3) 题意解析（避免混叠的最大采样周期）
- 设信号最高角频率上限 ω_max = 10^4 rad/s。
- 采样角频率 ω_s ≥ 2 ω_max ⇒ f_s ≥ ω_max/π ≈ 3183.099 Hz ⇒ T_e ≤ 1/f_s ≈ 0.000314 s。

解题结果
- 最大采样周期：T_e,max ≈ 0.314 ms。
- 建议取更快（如 0.1–0.2 ms）以便实现 PI 数字化与滤波。

## 3 Régulation numérique de température（RST）

问题摘要（见 C.16–C.19）：温控对象慢、允许很慢闭环；有输入端斜坡扰动 V(z) 与输出端正弦扰动 P(z)；要求：阶跃零静差（含 V 为斜坡时也零静差），且能拒斥 P 的特定频率；希望闭环比开环更慢（指定一阶目标，时间常数 5 小时）。

3.1 模型辨识与离散化

1) 题意解析（从阶跃响应读 k, τ）
- 假设连续开环：G(p) = k/(1 + τ p)^2。
- 从 C.17（阶跃响应）读：
  - s(∞) = k；
  - 形状为二阶串联一阶的二次积分形式 → 通过 63%/86%等拐点或斜率切线法估 τ。

解题步骤
- 在图上读取关键点（如拐点、斜率），按二阶同常数模型的阶跃模板估计 τ；
- 给出 k ≈ s(∞)。

2) 题意解析（ZOH 离散）
- 取 T_e = τ/5。
- 目标：求 G(z) = Z{B0(p) G(p)}。

解题步骤（给通用形式）
- 对 (1 + τ p)^{-2} 用已知 ZOH 表：离散二阶等效分母为 z^2 − 2 a z + a^2（a = e^{−T_e/τ}），分子按保持器矩形卷积得到线性组合（可引用表或推演）。
- 得：
  - a = exp(−T_e/τ) = exp(−1/5)；
  - G(z) = (b1 z + b0) / (z^2 − 2 a z + a^2)，b1, b0 由 τ, T_e, k 决定（可从标准表或数值工具获得）。

3.2 期望闭环模型与 M(z)

1) 题意解析（连续期望模型）
- 要求“更慢的一阶”：M_c(p) = 1/(1 + T_m p)，T_m = 5 小时 = 18000 s。

2) 题意解析（ZOH 离散）
- M(z) = Z{B0(p) M_c(p)} = (1 − b) z^{-1} / (1 − b z^{-1})，其中 b = e^{−T_e/T_m}。

解题步骤
- 已知 T_e（前选 τ/5，且 τ ≪ T_m），计算 b；
- 写出 M(z)。

3.3 RST 设计（拒斥与零静差）

基本结构（见 C.18/C.19）：闭环

H(z) = Y/W = (B(z) T(z)) / (A(z) S(z) + B(z) R(z))

设计要点
- 型别：为了阶跃零静差且输入端 V 为斜坡仍零静差，需要至少两个单位根：取 S(z) = (1 − z^{-1})^2 S_3(z)。
- 可补偿部分：将 G(z) 分解为 B^+, B^−, A^+, A^−。以 S_2(z) 补偿 B^+，R 与 S_3 完成极点配置。
- 目标：令闭环 H(z) = M(z)。

丢番图等式（标准）：

A(z) S(z) + B(z) R(z) = A_m(z)

B(z) T(z) = B_m(z)（若需要单位稳态增益）。

解题步骤
1) 分解：G(z) = B/A = B^+ B^− / A^+ A^−，稳定零/极点归入“+”，不稳定与纯延迟归入“−”。
2) 取 S(z) = (1 − z^{−1})^2 S_2(z) S_3(z)，其中 S_2(z) = A^+(z)/B^+(z)（补偿），S_3(z) 用于配置。
3) 设 R(z) 为与 S_3(z) 同阶或低一阶多项式，联立

A(z) (1 − z^{−1})^2 S_2(z) S_3(z) + B(z) R(z) = A_m(z)

求解 S_3, R。
4) T(z) 由 B(z) T(z) = B_m(z) 给出，或取以稳态匹配为准（例如令 H(1) = M(1)）。

稳定性与因果性检查
- S(z) 无不稳定零点（避免在单位圆外），控制器 C(z) = R/S 的极点应在单位圆内；
- 因果：deg(R) ≤ deg(S)。

3.4 去正弦扰动（频率已知）

已知卫星日照导致温度测点正弦扰动，周期 T = 90 min ⇒ 频率 f = 1/(90×60) Hz ⇒ 离散角频率 ω = 2π f；

题意解析
- 在输出通道的 P(z) 输入：分析 Y/P 传函，给出令 Y 对该频点无响应的条件。
- 方法：在 S(z) 或观测多项式 A0(z) 中嵌入相应的单位圆复零（z = e^{±jωT_e}），或令分母包含 (1 − 2 cos(ωT_e) z^{-1} + z^{-2}) 因子，使该频率被“陷波”。

解题步骤
- 计算 ω_d = 2π/(90×60) rad/s，φ = ω_d T_e。
- 在 S(z) 中乘以 (1 − 2 cosφ z^{-1} + z^{-2})，并相应提升 R(z) 与 T(z) 的阶次以保持因果。
- 重新解丢番图使 H(z)=M(z) 成立，同时达到 Y 对该频率的零响应。

3.5 关键结果与检查
- 闭环稳态：用最终值定理

lim_{k→∞} y[k] = lim_{z→1} (1 − z^{-1}) Y(z)

检查阶跃零静差（含 V 为斜坡时零静差 → 型别满足 2）。
- 稳定性：Jury 判据对闭环多项式；控制器因果性与有界性。
