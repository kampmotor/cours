# Examen 10 详细讲解与解答（Fibonacci 与不稳定一阶：Z、RST、响应分析）

- 题意概览：第一部分以 Fibonacci 序列为离散系统输出，推导其 Z 变换与由单位阶跃产生该输出的系统传递函数，并讨论用纯增益闭环无法稳定、转而采用 RST 综合并指定期望闭环模型；第二部分研究不稳定连续一阶对象，经 ZOH 离散化后用“响应堆”法（pile）综合稳定化控制器，分析首个采样间隔内的模拟输出、选择采样周期以达指定幅值、以及必然的过冲下界。

## 1. 第一部分：Fibonacci 与不稳定离散开环

Fibonacci 定义：$f_{n+2}=f_{n+1}+f_n$，$f_0=0,f_1=1$。应用“前移定理”（advance）：

- $\mathcal{Z}[f_n]=F(z)$，$\mathcal{Z}[f_{n+1}]=z F(z) - z f(0)$，$\mathcal{Z}[f_{n+2}]=z^2 F(z) - z^2 f(0) - z f(1)$。
- 代入递推得：$F(z)(z^2 - z - 1) = z \Rightarrow F(z)=\dfrac{z}{z^2 - z - 1}$。

令离散系统在单位阶跃输入下输出为该 $F(z)$：

- $U(z)=\dfrac{z}{z-1}$，故 $G(z)=\dfrac{F(z)}{U(z)}=\dfrac{z-1}{z^2 - z - 1}=\dfrac{(1 - z^{-1}) z^{-1}}{1 - z^{-1} - z^{-2}}$。
- 极点：$z_{1,2}=\dfrac{1\pm\sqrt{5}}{2}$，其中 $z_1\approx1.618$（不稳定），$z_2\approx-0.618$。

尝试用纯增益 $K$ 闭环（见图 C.33）稳定：闭环传递函数

$FTBF(z)=\dfrac{K(z-1)}{z^2 + (K-1)z - (K+1)}$。

Jury 第一条件 $D(1)>0$ 给出 $-1>0$，矛盾，说明任意 $K$ 都不能稳定该结构。

## 2. RST 综合与期望闭环模型

将 $G(z)$ 分解为可/不可消去部分：

$G(z)=\dfrac{B(z)}{A(z)}=\dfrac{B^+(z) B^-(z)}{(1-z^{-1})^{m} A^+(z) A^-(z)}$，其中

- $B^-(z)=(1 - z^{-1}) z^{-1}$，$A^+(z)=(1 - z_2 z^{-1})$（可消去），$A^-(z)=(1 - z_1 z^{-1})$（不可消去），$m=0$。

回路结构得 $F_{WY}(z)=\dfrac{B T}{A S + B R}$。

为对阶跃参考与扰动均实现零静差，取 $S(z)=(1 - z^{-1})^n S_1(z)$，需 $n=1$。

希望闭环模型为一阶+ZOH：

$M_{WY}(z)=\dfrac{1-\alpha}{z-\alpha}=\dfrac{(1-\alpha) z^{-1}}{1 - \alpha z^{-1}}$。

不补偿可消极点（按题面要求），令 $F_{WY}(z)=M_{WY}(z)$ 并化简为丢番图：

$(1 - z^{-1} - z^{-2}) S_1(z) + z^{-1} R(z) = 1 - \alpha z^{-1}$。

选最小阶：$S_1(z)=s_0$，$R(z)=r_0 + r_1 z^{-1}$。匹配得：

- $s_0=1$，$r_0=(1-\alpha)$，$r_1=1$。

同时取 $T(z)=(1-\alpha)$，故

- $S(z)=(1 - z^{-1})$，$R(z)=(1 - \alpha) + z^{-1}$，$T(z)=(1 - \alpha)$。

该设计实现单位静差为零、闭环一阶极点位于 $\alpha$，并保留不可消去的开环不稳定极点在分子/分母的适当位置以保证可实现性。

## 3. 第二部分：不稳定一阶对象的离散综合

连续对象：$G(p)=\dfrac{1}{1 - \tau p}$（$\tau>0$，不稳定）。ZOH 离散：

$G(z)=\dfrac{1 - a}{z - a}=\dfrac{(1 - a) z^{-1}}{1 - a z^{-1}}$，其中 $a=e^{T_e/\tau}>1$。

按响应堆法设方程：

$(1 - z^{-1}) A^{-} K(z) + B^{+}(z) B^{-}(z) L(z) = 1$，即

$(1 - z^{-1})(1 - a z^{-1}) K(z) + (1 - a) z^{-1} L(z) = 1$。

最小阶解：$K(z)=k_0$，$L(z)=l_0 + l_1 z^{-1}$，匹配得

$k_0=1,\ l_0=\dfrac{1+a}{1-a},\ l_1=\dfrac{-a}{1-a}$。

控制器：

$C(z)=\dfrac{A^{+}(z) L(z)}{(1 - z^{-1}) K(z)}=\dfrac{l_0 + l_1 z^{-1}}{1 - z^{-1}}$。

- 首个采样间隔内的模拟输出：系统受幅值为 $l_0$ 的阶跃驱动（采样 0 到 $T_e$），连续输出

$y(t)=l_0 \big(1 - e^{t/\tau}\big),\ 0<t<T_e$。

- 设定 $y(T_e)=3$（单位阶跃参考），利用 $a=e^{T_e/\tau}$ 有

$y(T_e)=1 + e^{T_e/\tau}=1 + a$，故 $a=2\Rightarrow T_e=\tau\ln 2$。

- 过冲下界：$y(T_e)=1+a\ge 2$，因此至少 100% 超调。

## 4. 要点总结

- Fibonacci 的 Z 变换与由阶跃生成该序列的离散系统 $G(z)$ 可直接推导；其不稳定极点使得“仅加增益”无法稳定，需要结构化控制（RST）。
- RST 通过设定 $S=(1 - z^{-1})S_1$ 实现阶跃参考与扰动零静差；选择不补偿某些极点以保证实现性，丢番图解给出最小阶 $R,S,T$。
- 不稳定一阶对象经 ZOH 离散后可用响应堆法综合得到可实现的差分控制器；首周期内连续域输出由等效阶跃决定，可据此反推采样周期以达目标幅值，同时显示过冲的不可避免下界。
