# Examen 8 详细讲解与解答（直流电机速度与位置控制：比例、双环、PI、数字）

- 题意概览：以直流电机为对象，先建模得到速度通道传递函数，分析开环极点与波德图；再用比例控制闭环，用Routh判断稳定并计算稳态值与阻尼；加入电流内环改善阻尼；然后用两时间常数近似与Ziegler–Nichols给出模拟PI；最后在数字域用已给G(z)做稳定性范围与静差分析，并讨论带积分的数字控制器的因果性。
- 已知参数：`J=2 kg·m^2, K=1 V·s·rad^-1, R=1 Ω, L=0.05 H, f=0`，传感器系数 `K'=0.1 V·s·rad^-1`。

## 1. 建模与分析

由机电耦合方程：

- 电枢：`u = R i + L di/dt + e`，反电势 `e = K ω`
- 机械：`J dω/dt = γ + γ_p - f ω`，电磁转矩 `γ = K i`

消去 `i, e, γ` 得速度通道：

$G(p)=\dfrac{Ω(p)}{U(p)}=\dfrac{K}{J L p^2 + J R p + K^2}$。

这是一标准二阶最小相位系统。按给定数值，开环极点约在 `p1≈-19.5` 与 `p2≈-0.513`。波德图呈低频平坦、两拐点后以 −40 dB/dec 下落，相位从 0° 逐步降至约 −180°。

---

## 2. 比例控制与稳态/阻尼

闭环采用速度反馈与比例器 `C(p)=A`、传感器 `K'`：

$H(p)=\dfrac{Ω(p)}{Ω_c(p)}=\dfrac{A K}{J L p^2 + J R p + K^2 + A K K'}$。

- Routh 稳定性：系数均正且满足 `K^2 + A K K' > 0`，对任何 `A>0` 均稳定（题面给出条件式与结论）。
- 最终值（例 `A=1`，单位阶跃指令）：

$\lim_{t\to\infty} ω(t)=\dfrac{A K}{K^2 + A K K'} \overset{A=1}{=} \dfrac{1}{K+K'} \approx 0.9091$；传感器输出为其十分之一约 `0.09091`。

- 误差2%所需增益：题面给出 `A≈4990`。
- 规格化二阶式与阻尼：将$H(p)$写成

$H(p)=\dfrac{\alpha}{1 + \dfrac{2 m}{\omega_0} p + \dfrac{1}{\omega_0^2} p^2}$，

其中 $\omega_0=\sqrt{\dfrac{K^2 + A K K'}{J L}}$，$m=\dfrac{1}{2}\dfrac{J R}{\sqrt{J L (K^2 + A K K')}}$。

数值示例给出 `ω0≈70`、`m≈0.14`：阻尼很小，超调大、整定慢。

---

## 3. 双环（速度外环+电流内环）

加入电流内环，等效在速度通道对电枢电阻产生 `R→R−AB` 的作用：

$H(p)=\dfrac{\frac{A K}{K^2 + A K K'}}{\frac{J L}{K^2 + A K K'} p^2 + \frac{J (R-AB)}{K^2 + A K K'} p + 1}$。

因此只影响一阶项系数，从而只改变阻尼项 $\dfrac{2m}{\omega_0}$，与题面一致。设定希望超调 5%，对应阻尼比近似 $m\approx0.707$；由

$m=\dfrac{1}{2}\dfrac{J (R-AB)}{\sqrt{J L (K^2 + A K K')}}$，解得 `B≈−8×10^-4`（题面数值）。注意符号意味着内环校正方向需按具体实现确认。

对单位阶跃扰动转矩 `γ_p(t)` 的静态影响：速度为类型 0 对外加力矩阶跃存在非零静差，需积分环节或力矩前馈方可消除，题面留作计算练习。

## 4. 模拟 PI（两时间常数+小延时近似，ZN 整定）

对象两时间常数近似：

$G(p)=\dfrac{\beta}{(1+\tau_1 p)(1+\tau_2 p)} \simeq \dfrac{1}{(1+1.95 p)(1+0.05 p)}$，将快速支路近似为微小延时：

$G(p)\approx \dfrac{e^{-\tau_2 p}}{1+\tau_1 p}$，其中 `τ2≈0.05 s` 为近似的纯延时，近似有效频段至 $f_c=\dfrac{1}{2\pi\tau_2}$。

依据阶跃响应（S 形）读取 ZN 参数，题面给出：

- 比例增益 `K ≈ 0.9·T/(E·τ)`，代入数值得 `K ≈ 385.88`
- 积分时间 `Ti = 3.3 τ2 ≈ 0.165`

因此模拟 PI 可取：`C_PI(p) = K (1 + 1/(Ti p))`。

## 5. 数字调节（Jury 稳定域、静差、带积分的因果性）

已给 ZOH 后模型（`Te=0.05 s`）：

$G(z)=\dfrac{0.09179\,(1+0.7181 z^{-1})\,z^{-1}}{(1-0.9747 z^{-1})(1-0.3774 z^{-1})}$。

- 比例数字闭环：

$H(z)=\dfrac{A G(z)}{1 + A K' G(z)}$。按题面 Jury 检验得稳定区间 `−0.99 < A < 95.77`（取 `A>0` 则 `0 < A < 95.77`）。

- 静差下界：对最大稳定增益 `A≈95.77` 计算最终误差约 `ε(∞)≈0.01`（题面计算过程给出）。

- 引入带积分的数字控制器：

$C(z)=K \dfrac{(1-b_0 z^{-1})(1-b_1 z^{-1})(1-b_2 z^{-1})}{(1-a_1 z^{-1})^l (1-a_0 z^{-1})}$。

取 `a1=1` 在分母形成单位根即离散积分器，可实现阶跃指令零静差。但需检查因果性与实现可行性：当分母阶数高于分子总阶数且伴随多重积分（大 `l`），若未增加足够前向零或纯延时，可能导致非因果或实现不稳定。题面结论给出：该配置在所给设定下“非因果”。

---

## 6. 要点总结

- 开环为二阶最小相位系统，比例闭环在任意正增益下稳定，但阻尼偏小，易超调大、整定慢。
- 加入电流内环仅改变阻尼项，可通过选择 `B` 显著提高阻尼、减少超调。
- 用两时间常数近似把快支路当作小延时，按 ZN 得出模拟 PI 参数，能改善稳态与动态。
- 数字域下用 Jury 检验稳定增益范围；在纯比例下静差不可达 0；若要零静差需引入离散积分，但需兼顾因果性和实现约束。
