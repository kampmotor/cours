## 1. 部分一：零/极点补偿现象

### 1.1 稳定极点补偿
结构：输入 w 经零$(1 - a z^{-1})$，后接极点 $1/(1 - b z^{-1})$。完美补偿时 a=b=0.5。

- 差分式：$u_k = w_k - a w_{k-1}; y_k = u_k + b y_{k-1}$。
- 单位阶跃 $w_k=1$：完美补偿时 $u_0=1, u_{k≥1}=0.5；y_0=1, y_{k≥1}=1$ —— 补偿消除了极点的动态（瞬时达到稳态）。
- 非完美 $(a=0.5, b=0.4)$：y 序列：$1, 0.9, 0.86, 0.844, 0.838...$收敛到稳态增益 $(1 - a)/(1 - b)=0.5/0.6≈0.8333$。轻度残余动态（几何衰减）。

要点：
- 稳定极点被单位圆内零精确补偿 → 快速响应但对噪声更敏感（失去滤波作用）。
- 不完全补偿 → 剩余极点保留部分延时/滤波能力。

### 1.2 不稳定极点补偿（零在单位圆外）
设 a=2（零），b=1.9（极点近不稳定）。

- $u_k = w_k - a w_{k-1}$ 给阶跃：$u_0=1, u_{k≥1}=-1$。序列不发散（$常值$），说明单个外单位圆零对输入差分可能形成有界输出来抵消增长模式，但并不说明整体系统稳定。
- $y_k = u_k + b y_{k-1}$：产生序列 1, 0.9, 0.71, 0.349, -0.337,... 绝对值开始向更大振荡趋势（将继续反向扩展），显示组合后仍呈不稳定行为（极点大于 1）。

要点：
- 补偿外单位圆“极点”或使用外单位圆零试图抵消不稳定模式通常失效（参数轻微偏差即发散）。
- 理论结论：不应在实践中依赖对不稳定极点的精确零补偿进行稳定化，需将极点移入稳定区域。

## 2. 部分二：一阶对象 + 传感器的 PI 综合
对象与传感器：$G(p)=1/(1+τ p), H(p)=2/(1+τ_c p)，τ=1 s, τ_c=0.2 s$。

### 2.1 模拟 PI（Ziegler–Nichols 延时近似）
将传感器小时间常数近似为纯延时: $(1+τ_c p)^{-1} ≈ e^{-τ_c p}$ → 读图参数: T=1, τ_s=0.2, 步阶高度 E=2 → a=E=2。
ZN：$K=0.9/(a τ_s)=0.9/(2·0.2)=2.25$；Ti=3.3 τ_s≈0.66。控制器 C_PI(p)=2.25 (1 + 1/(0.66 p))。

### 2.2 数字采样与 Zdan 分解
采样目标：闭环速度接近传感器动态 → 选 Te≈τ_c/5=0.04 s（给出示例）。题面后续固定 Te=0.1 s 形成离散模型：
$G(z)=(1 - a) z^{-1} / (1 - a z^{-1}), a=e^{-Te/τ}=e^{-0.1}=0.9048$。
$GH(z)=0.04117 (1+0.8189 z^{-1}) z^{-1} / [(1 -0.9048 z^{-1})(1 -0.6065 z^{-1})]$。

闭环目标：
- 一阶期望: $M(z)=((1-b) z^{-1})/(1 - b z^{-1}), b=e^{-Te/τ_c}=e^{-0.1/0.2}=e^{-0.5}=0.6065$。
- 零静态斜坡误差 → 需要二重积分器 $(1 - z^{-1})^{-2}$。
- 补偿 G(z) 可消部分：C_2(z)=(1 -0.9048 z^{-1})/0.09516。
- 剩余形状由 $C_3(z)=Δ_1(z)/Δ_2(z)$ 解丢番图：
$(1 - z^{-1})^2 (1 - b z^{-1}) Δ_2 + 0.4326 (1 +0.8189 z^{-1}) z^{-1} Δ_1 = (1 - b z^{-1})。$
$解：Δ_2=1+0.6529 z^{-1}; Δ_1=3.1139 -3.7317 z^{-1} +1.1178 z^{-2}。$
$最终 C(z)=C_1 C_2 C_3：$
$C(z)= (1/ (1 - z^{-1})^2) * ((1 -0.9048 z^{-1})/0.09516) * (Δ_1/Δ_2)。稳定且因果；阶跃参考下闭环输出最终值受传感器增益影响为 0.5，跟踪误差为零（目标一阶模型与测量链串联）。$

要点：Zdan 分解将性能需求拆为：积分个数（阶跃→1, 斜坡→2），补偿可控稳定零极点，剩余动力学由丢番图精确匹配。

## 3. 部分三：双积分器系统与离散响应堆综合
连续对象：G(p)=1/[p^2 (1 + τ p)]。

### 3.1 PID 稳定性条件
PI 不可稳定：闭环缺少 p^2 正项（或 Routh 条件破坏）。PID 增加 k_d 恢复四阶完整多项式：τ p^4 + p^3 + k_d p^2 + k_p p + k_i。Routh 必要条件：
- k_d - τ k_p > 0
- k_i > 0
- (τ k_p^2 + k_i)/(k_d - τ k_p) > 0 （隐含 k_i > -τ k_p^2, 与前条件结合）
提供稳定设计可行范围的三不等式集合。

### 3.2 离散化与“响应堆”式综合
$Te=τ → a=e^{-1}=0.3679。离散模型：$
$G(z)= (b_2 z^2 + b_1 z + b_0)/(z^3 - (2 + a) z^2 + (1 + 2a) z - a) = (…)/( (z -1)^2 (z - a) )，确认有双单位根与一稳定根 a。$
目标：阶跃响应为“响应堆”（deadbeat/ pile）且能拒绝斜坡扰动，需要结构：
$1 - F(z) = (1 - z^{-1}) (1 - z^{-1})^2 K(z) = (1 - z^{-1})^3 K(z)$
$F(z)=B(z) L(z)$。
$丢番图：(1 - z^{-1})^3 K + B L = 1。$
$阶次判定：deg B=3（含 z^{-1} 与两零因式展开），deg (1 - z^{-1})^3=3 → 正规方程：deg K=2, deg L=2。$
$已给 K'(z)=1 +1.8396 z^{-1} +0.3346 z^{-2} → 误差多项式对阶跃：ε(z)=(1 - z^{-1})^2 K'(z)=1 -0.1604 z^{-1} -2.3446 z^{-2}（限速误差结构）$
对斜坡与抛物输入的误差表达式展示积分阶数与未利用纯积分环节的影响

要点：多重积分 (1 - z^{-1}) 功能区分：补偿被动极点 vs 引入跟踪阶次能力；阶次分配由丢番图正规度决定。

## 4. 部分四：RST 结构等价与闭环模型实现
$开环：G(z)=(z - 0.8)/(z (z - 0.6)) = B/A。$
$分解：B^+=1 - 0.8 z^{-1}, B^-=z^{-1}; A^+= (1 -0.6 z^{-1})^2, A^-=1。$
$目标闭环：H(z)=B_m/A_m = 3.1 (1 -0.5 z^{-1}) z^{-1} / (1 +0.5 z^{-1} +0.05 z^{-2})。$

### 4.1 结构等价
原结构带“前馈与积分”：\~R, \~S, \~T 与 β（改进跟踪）；等价标准 R,S,T：
$S= \~S (1 - z^{-1}); R= \~R (1 - z^{-1}) + \~T (1 - β z^{-1}); T= \~T (1 - β z^{-1}).$

### 4.2 选择与简化
$假设：\~S= \~S_1 B^+, \~R= \~R_1 A^+, \~T= \~T_1 A^+。$
$由于 B^-=z^{-1}, A^-=1，简化闭环：$
$H(z)= z^{-1} \~T_1 (1 - β z^{-1}) / [ \~S_1 (1 - z^{-1}) + z^{-1}( \~R_1 (1 - z^{-1}) + \~T_1 (1 - β z^{-1}) ) ].$
$匹配分子：\~T_1=3.1, β=0.5。$
$匹配分母：得到 \~S_1=1, \~R_1=-1.6。$
最终：
$S(z)= (1 - z^{-1}), R(z)= -1.6 (1 - z^{-1}) + 3.1 (1 - 0.5 z^{-1}), T(z)=3.1 (1 - 0.5 z^{-1}).$
可化简并编写差分控制律：
U(z)= (T(z) Y_c(z) - R(z) Y(z))/S(z)。
阶跃与同阶扰动下含单积分 (S 含 1 - z^{-1}) → 静差零；扰动阶跃亦被抑制。

## 5. 关键结论与答题要点
- 稳定极点零补偿：完美补偿加速响应但牺牲鲁棒性；不完美补偿保留部分动态与滤波。
- 不稳定极点补偿风险：理论上“精确”抵消极点不可靠，微小偏差导致发散，应采用稳定化而非直接零补。
- Zdan 分解：性能需求 → 积分个数 (阶跃/斜坡) + 可补偿部分 (稳定零极点) + 丢番图解决剩余闭环形状。
- 双积分器：PI 不足以稳定，需 PID 补 p^2 项；离散响应堆设计以高阶积分矩阵实现多阶跟踪与扰动抑制。
- RST 等价转换：前馈与积分结构可回写到标准 R,S,T；通过分子/分母匹配直接算出常数型 \~R_1, \~S_1, \~T_1。
- 考试答题策略：先写差分或传递函数结构→列出丢番图→给阶次理由→匹配系数→总结闭环特性 (极点位置、静差、扰动抑制类型)。
